# Pi, e, square Root 2 finder

이 서비스는 데스크탑 웹 / 모바일 웹에서 동작하는 웹서비스로
Pi, e, square root 2 를 외우는 기능과 거기서 특정 문자열을 찾는 기능이 있어


## Pi 를 외우는 기능
소숫점 이하부터 외우는 옵션과, 3. 부터 외우는 옵션이 있고, 한자리-9자리씩 외울지 정하는 옵션도 있어
한자리씩 외울 땐 계산기 같은 키패드가 나와서 숫자 하나씩 계속 입력해
두자리-9자리 씩 할 땐 객관식이야 화면 크기에 맞춰서 보기가 나오면 될 것 같아.

사용자가 선택한 옵션은 LocalStorage로 남으면 좋겠어

최대 천자리까지 외울 수 있으면 좋겠어

몇 자리까지 했는지, 걸린 시간은 얼마인지, 기록하고 
이름도 기록 가능해야돼. 
사용자 별로 최초에 랜덤으로 별명을 부여해. 이건 꾸밈어 + 역대 유명한 수학자 이름 (돌아가신 분) 이 기본 설정이고, 옵션에서 바꿀 수 있어
기록은 로컬스토리지 사용해서 남기고, 서버에 올리는 건 추후 할꺼야.

## Pi 에서 특정 문자열 찾는 기능
이건 Pi 백만자리까지에서 최대 8자리의 수가 있는지 찾는 기능이야.
최초 위치, 몇 번 나오는지 알려주면 돼 (팝업 말고 화면 밑에 표시)
혹시 너무 오래 걸리면 추후 기능을 조정할게
